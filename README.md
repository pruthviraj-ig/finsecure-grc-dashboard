# FinSecure GRC Dashboard

A web-based Governance, Risk, and Compliance (GRC) dashboard built to simulate risk monitoring and compliance oversight for a fictional UK fintech company, FinSecure Ltd.

## Features
- Executive overview cards
- Risk analytics dashboard
- Risk heatmap
- Assets → Risks → Controls flow view
- Risk register
- Control tracking
- Compliance alignment with GDPR, ISO/IEC 27001, and NIST Cybersecurity Framework
- Dark/light mode toggle

## Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

## Project Purpose
This project was created to demonstrate practical GRC concepts including:
- risk identification
- risk scoring
- control effectiveness tracking
- residual risk monitoring
- compliance mapping

## Data Source
The dashboard is based on a fictional risk assessment for FinSecure Ltd, created as part of a GRC portfolio project.
